第三方页面 demo (v5 通用工具模型)

本页面演示 浮窗机器人 SDK v5 的通用工具模型: 持久工具(init 传入,所有会话生效) + 临时工具(追加注入,仅当前会话生效,新会话自动失效)。

订单录入区(SDK 智能录单会把字段填到这里)

v5 通用工具模型演示

持久工具:init 时传入的 submit_form + 内置 change_skin。每个新会话自动注入,不会丢失。

临时工具:点击下方按钮追加,仅当前会话生效。点"新会话"后临时工具自动失效。

changeSkin: 开启 | 临时工具: 无
